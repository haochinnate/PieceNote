# 文法

### ～かいがあっと

- [連結](https://www.youtube.com/watch?v=tYF1mFnqzzw)
- 努力のかいがあって、大学に合格しました。
- 努力：どりょく、合格：ごうかく

#### 意思
- ~~~有努力, 有好的回報
- ます型, 去掉ます, 值得做....行為

#### 接續

- N　→　努力のかい（が）あって
- V　→　頑張った　かい（が）あって
- V　→　作り　がいがある　（作り：つくり）

#### 例句

- 練習したかいがあって、優勝することができた。
- 練習：れんしゅう；優勝：ゆうしょう

- リハビリのかいがあって、歩けるようになった。
- リハビリ：復健；歩ける：あるける

- 夫はおいしそうにご飯を食べるので作りがいがある。
- 夫：おっと

- 急いだ／残業した　かいがあって、間に合いました。
- 急いだ：急いだ；残業：ざんぎょう；間に合い：まにあい

- 毎日　ジョギングした／筋トレした　かいがあって瘦せました。
- 毎日：まいにち；瘦せます：やせます；ジョギング：跑步；筋トレ：健身

### ～だけまし

- [連結](https://www.youtube.com/watch?v=6w5UWsH7wSo)
- コロナで大変だけど、仕事があるだけましだ。
- 大変：たいへん、仕事：しごと

#### 意思

- 不好的情況, 但跟其他人/其他情況比起來 已經算不錯了

#### 接續

- V/A/Na(普通形) + だけまし
- Na だ　→　な

#### 例句

- 会社の食堂はおいしくないけど、安いだけましだ。
- 食堂：しょくどう

- 事故で車は壊れたけど、怪我がなかっただけましだ。
- 事故：じこ、壊れた：こわれた、怪我：けが

- 今日は寒いけど、雨が降らないだけましだ。
- 寒い：さむい、降らない：ふらない


### ～ものなら

- [連結](https://www.youtube.com/watch?v=TSdqk4HV4dU)
- 辞められるものなら、この会社を辞めたい
- 辞める：やめる

#### 意思

- 針對很難實現, 但是想這樣做的願望

#### 接續

- V(可能形) + ものなら
- 會話: ～もんなら

#### 例句

- あのときに戻れるものなら、今すぐ戻りたい。
- 戻り：もどり：回到
- やり直せるものなら、もう一度やり直したい。
- やり直す：重新來做
- 子どもが病気で代われるものなら代わってあげたい。
- 病気：びょうき、代わる：交換

### ～ともかく

- [連結](https://www.youtube.com/watch?v=zHPvEMKfFAU)
- ここの食堂の料理は見た目はともかく味はおいしい。
- 食堂：しょくどう、見た目：みため、味：あじ

#### 意思

- 先不談...重點是...

#### 接續

- ～はともかく～（は／が）

#### 例句

- あの俳優は顔はともかく、演技が素晴らしい。
- 俳優：はいゆう、顔：かお、演技：えんぎ、素晴らしい：すばらしい
- この服はデザインはともかく、色がよくない。
- 服：ふく、デザイン、色：いろ
- 私は英語はともかく、日本語が苦手だ。
- 苦手：にがて

### ～くせして

- [連結](https://www.youtube.com/watch?v=tR8eLqY9mbE)
- 大学生のくせして、そんなことも知らないの？

#### 意思

- 立場, 身分 身為...卻...

#### 接續

- 普通形 + くせして
- Nのくせして
- Na形 + なくせして
- 會話: ～くせに

#### 例句

- あの俳優、奥さんがいるくせして不倫しているらしい。
- 俳優：はいゆう、奥さん：おくさん
- 男のくせして、それだけしか食べないの？
- 男：おとこ
- のび太のくせに生意気だぞ。
- 生意気：なまいき
- 彼女は自分が悪いくせして文句ばかり言っている。
- 文句：もんく

### ～がたい

- [連結](https://www.youtube.com/watch?v=PFBFw2-tF68)
- 修学旅行は学生にとって忘れがたい思い出です。
- 修学旅行：しゅうがく　りょこう
- 忘れ：わすれ

#### 意思

- ～するのが難しい
- 很難...

#### 接續

- Vます＋がたい

#### 例句

- 今回の結果は受け入れがたい。
- 結果：けっか、受け入れる：うけいれる
- 詐欺は本当に許しがたい行為です。
- 詐欺：さぎ、許します：ゆるします、行為：こうい
- 彼の考えは理解しがたいが、論理的には正しい。
- 考え：かんがえ、理解：りかい、論理：ろんりてき：邏輯、正しい：ただしい

### ～っこない

- [連結](https://www.youtube.com/watch?v=CqwvE9XNhZU)
- こんなにたくさんの単語一日で覚えられっこない。
- 単語：たんご、覚えられる：おぼえられる

#### 意思

- 絕對に～ない／～はずがない
- 絕對不會怎樣....

#### 接續

- Vます形 去ます + っこない

#### 例句

- 今の実力では試験に受かりっこない。
- 実力：じつりょ、受かり：うかり
- テニスで鈴木君に勝てっこない。
- 勝て：かて
- 人は他人の気持ちなんて分かりっこない。
- 他人：たにん

### ~かなには

- [連結](https://www.youtube.com/watch?v=fZzXELTga38)
- 試合に出るからには優勝したい
- 試合（しあい）、優勝（ゆうしょう）

#### 意思

- 既然...就...

#### 接續

- 普通形＋からには
- 名詞な形＋であるからには

#### 例句

- 付き合うからには結婚を考えている
- 付き合う（つきあう）、考えて（かんがえて）
- 留学するからには日本語が話せるようになりたい
- この会社で働くからには　会社のルールを守ってください

### ~たまらない

- [連結](https://www.youtube.com/watch?v=2RAn_gpgdgk)
- この部屋は暑くてたまらない
- 暑くて（あつくて）

#### 意思

- 非常...(無法忍耐的程度)

#### 接續

- て形＋たまらない
- い形　→　くてたまらない
- な形　→　でたまらない

#### 例句

- 試合に負けたのが、悔しくてたまらない
- 負けた（まけた）、悔しい（くやしい）
- 井上先生の授業は眠くてたまらない
- 井上（いのうえ）、授業（じゅぎょう）、眠くて（ねむくて）
- 日本はどうしてあんなに居酒屋があるのか不思議でたまらない
- 居酒屋（いざかや）、不思議（ふしぎ）

### 其他

- 〜だって
 - そのくらいは小学生 __だって__ 知っているはずだ
- 〜でしかない
 - スマホはただの連絡手段 __でしかない__
- 〜でなくてもよければ
 - 今日 __でなくてもよければ__ 買い物に付き合える
- 〜として
 - 彼は歌手 __として__　有名になった
- 〜としては
 - 私 __としては__　あなたの意見に賛成することができません
- 〜において
 - 人生　__において__　最も重要だと思うことは何ですか
- 〜にかけて
 - この仕事は夏休み　__にかけて__ よくに忙しくなる
- 〜に比べて
 - 日本 __に比べて__ ベトナムの方が物価が安い
 - 彼女は仕事  


# 文法篇

## 0. 基本詞性

### 普通形

- 動詞
  - 現在肯定, 否定: 読む, 読まない
  - 過去肯定, 否定: 読んだ, 読まなかった
- イ形容詞
  - 現在肯定, 否定: おいしい, おいしくない
  - 過去肯定, 否定: おいしかった, おいしくなかった
- ナ形容詞
  - 現在肯定, 否定: 親切だ, 親切ではない
  - 過去肯定, 否定: 親切だった, 親切ではなかった
- 名詞
  - 現在肯定, 否定: 先生だ, 先生ではない
  - 過去肯定, 否定: 先生だった, 先生ではなかった

### 動詞詞性

- 動詞辭書形: 読む
- 動詞ない形: 読まない
- 動詞ます形: 読みます
- 動詞て形: 読んで
- 動詞ている形: 読んでいる
- 動詞た形: 読んだ
- 動詞假定形：読めば
- 動詞意向形: 読もう

## 1. 助詞

### 強調助詞

1. 〜が
  - 用於敘述某物的內容或加強語氣的敘述
  - 名詞 + が
  - 蛇は足がありません。
    - へび

2. 〜も
  - 〜之多, 甚至～, 強調程度之高, 數量之大, 或事件之不尋常
  - 名詞 + も
  - 今日は忙しくてご飯を食べる時間もなかった。
  
3. 〜(助詞) + は
  - 表示語氣上的積極性. 強調時、「は」可以接在「に」「で」、「へ」之下使用 ◦
  - 名詞 + には / では / へは
  - MRTの中では物を食べてはいけません。

4. 〜(で)さえ
  - 中文解釋為「連〜」。舉某一極端例子, 強調比此例更高或更低程度的事物
  - 名詞 + (で)さえ
  - 父は仕事が忙しく, 日曜日さえ休めない。

5. 〜まで
  - 「甚至～」, 強調同類事物的增添
  - 名詞 + まで
  - 最近は老人までダンスを習っている。
    - ろうじん

6. 〜だけ
  - 「盡量~」「盡管~」, 用來強調盡量或是盡可能的程度
  - 名詞, イ形容詞, ナ形容詞(な), 動詞辭書形 + だけ
  - できるだけ今月中に借金を返します。
    - こんげつ, ちゅう, しゃっきん

7. 〜こそ
  - 「才~」、「正是~」, 說話者以肯定的態度強調某事物時使用
  - 名詞 + こそ
  - 京都こそ日本を代表する町です。
    - だいひょう

### 限定助詞

1. 〜で
  - 「在~之中」,「就〜」, 用於時間與空間事物及人物的限定表現
  - 名詞 + で
  - 二人でこの仕事をやろう。

2. 〜だけ
  - 「只有~」。 用於提示限定的對象、範圍、程度、性質、數量等。
  - 名詞 + だけ
  - 彼女だけがわたしのいい友達です。

3. 〜しか〜(ない)
  - 「只有~」。用於提示(限定)除此之外沒有其他的選擇, 文末以否定表現結束。
  - 名詞+ しか~（ない）
  - 教室には学生が3人しかいません。
    - きょうしつ, がくせい

4. 〜さえ
  - 「只要~」。 用於如果限定的條件成立的話, 就必然會發生的事實。通常與條件句型「さえ〜ば」或「さえ〜なら」搭配表現。
  - 名詞 + さえ + 條件句型
  - 体さえ丈夫なら, どこにでも遊びに行きたい。
    - からだ, じょうぶ

5. 〜ばかり
  - 「光~」,「只~ 」, 「淨是~ 」。用於提示同樣的 ( 限定的)東西或狀態很多, 句子通常含有負面的評價 o
  - 名詞/動詞て形 + ばかり
  - 弟はテレビばかり見ている。
    - おとうと

6. 〜までに
  - 「在~之前」, 用於提示限定的時間期限, 表示動作必須在此之前完成
  - 名詞 + までに
  - 3時までに会社に戻らなければならない。
    - もどらない

### 並列助詞

1. 〜も
  - 「也~ 」, 用於同類事物的並列。
  - 名詞 + も
  - 時間もお金もないので、旅行には行けない。

2. 〜と
  - 「和」, 用於同性質事物的並列(全部表現)。不能當列舉
  - 名詞 + と
  - かばんの中に鍵と携帯と財布があります。
    - かぎ

3. 〜や〜や
  - 「〜啦〜 」、「〜或〜」。 用於同性質事物的列舉 （部分表現）, 暗示除了舉出的事物之外, 還有許多類似的事物
  - 名詞 + や
  - かばんの中に鍵や携帯や財布などがあります。

4. 〜とか〜とか
  - 「〜呀〜呀」, 「又〜又〜」, 用於同性質事物的列舉
部分表現, 暗示除了舉出的事物之外, 還有許多類似的事物
  - 名詞/動詞辭書形 + とか
  - 時々は散歩をするとか映画を見るとか,　気分転換したほうがいいですよ。
    - ときどき, さんぽ, きぶんてんかん

5. 〜やら〜やら
  - 又...又..., 用於多項事物中舉出兩項為例, 暗示除此之外, 仍有許多同類的事物。
  - 名詞/イ形容詞/動詞辭書形 + やら
  - みんなに祝ってもらい、うれしいやら恥ずかしいやら、戸惑っています。
    - いわって, とまどって

### 程度助詞

1. 〜ほど
  - 比喻某狀態(動作)發生時的程度, 到如此的..., 到那樣的...
    - 動詞普通形 + ほど
    - 今度の台風は屋根が飛ぶほど風が強かった。
      - たいふう, とぶ
  - 兩者比較時的程度提示, 沒有那麼...
    - 名詞 + ほど + 否定表現
    - 英語は日本語ほど上手ではありません。
  - 程度的基準提示之用, 沒有比...更...
    - 名詞 + ほど
    - 彼ほど優しい人はいない。
  
2. 〜など
  - ～之類的, 像～這樣的. 用於說話者在心理上覺得某事物不重要
  - 或有些許輕視(厭煩)的感覺
  - 或是對自己的事物表示謙虛的敘述語法
  - 名詞/動詞辭書形 + など
  - そんな結果になるなどとは考えてもみなかった。
    - けっか

3. 〜なんか/〜なんて
  - ～之類的, 怎麼會～, 語氣比～など更強烈, 更不客氣
  - 名詞 + なんか, 名詞/動詞辭書形 + なんて
  - 大人なのに電車の中で漫画を読むなんて, みっともない。
    - おとな

4. 〜くらい/〜ぐらい
  - 比喻某狀態(動作)發生的程度, 簡直..., 像...
    - 動詞普通形 + くらい/ぐらい
    - 今朝の地震は本棚の本が落ちるぐらい大きかった。
      - けさ, ほんだな, おちる
  - 作為兩者比較時, 動作狀態程度提示, 與...相同, 和...一樣
    - 動詞辭書形, イ形容詞, 名詞, ナ形容詞な + くらい/ぐらい
    - 私は姉ぐらい背が高いです。
      - あね
  - 表示程度之最高或最低, 沒有比...更..., 至少還... 
    - 名詞 + くらい/ぐらい
    - タバコぐらい健康に悪いものはない。
    - 私だって掃除ぐらいしますよ。
      - そうじ

## 2. 具有助詞功能的文法句型

### 方法, 手段, 原因的相關句型

1. 〜によって, 〜により
  - 「因～」,「透過～」, 提示說話者做出判斷的方法或依據。
  - 名詞 + によって/により
  - 交通事故によって, 電車が遅れました。
    - こうつう, じこ

2. 〜によれば/〜によると
  - 「據〜」,「根據〜」, 提示資訊, 消息的來源
  - 名詞 + によれば/によると
  - 彼女の手紙によれば, 来月アメリカへ行くということだ。
    - てがみ, らいげつ

3. 〜を通じて
  - 「透過~」,「經由~」, 提示說話者透過某種方法(人/事/物)以進行或完成動作的表現
  - 名詞 + をつうじて
  - 友達を通じて、彼と知り合った。

4. 〜から
  - 以「AからB」的巨型出現, A表示原因, B則表示結果
  - 名詞 + から
  - ちょっとした不始末から、大騒ぎになってしまった。
    - ふしまつ

### 內容、對象的相關句型

1. 〜について
  - 「關於~」、「針對~」。就人物的言語或行為內容加以深入（深度） 說明的語法。
  - 名詞 + について
  - 来月卒業旅行について、 会議を開きます。
    - そつぎょう, りょこう

2. 〜に関して
  - 「關於~ 」, 「有關~」。 就前項的(名詞)事件、問題做出後項的回應。比「~について」的語法更為正式。
  - 名詞 + にかんして
  - 最近、経済学に関しての本を読んでいます。
    - けいざい, がく

3. 〜に対して
  - 「對～」,「向～」。用於對前項的名詞, 做出後項的行為或態度的回應
  - 名詞 + にたいして
  - 私の質問に対して, 詳しく説明してくれました。
    - しつもん, くわしい

4. 〜に応えて
  - 「回應～」, 針對他人的期待, 要求等做出回應行為時, 所使用的語法
  - 名詞 + にこたえて
  - 学生の願いに応えて, 図書館の開館時間が１１時まで延長されました。
    - かいかん, えんちょう

### 立場, 資格的相關句型

1. 〜上で
  - 在...方面看來。通常用來表達說話者對此做出的判斷
  - 名詞の/動詞辭書形 + うえで
  - 数字の上では, 今月の業績はよかったと言えます。
    - すうじ, こんげつ, ぎょうせき

2. 〜にとって
  - 對...而言。通常用來表達某事物對主語人物的影響。
  - 名詞 + にとって
  - わたしにとって, 犬も家族の一員です。
    - いちいん

3. 〜として
  - 以...的名義, 以...的身份
  - 名詞 + として
  - 大学の先生として, テレビで活躍しています。
    - かつやく

4. 〜にしたら
  - 以...立場來說。是說話者站在當事人的立場, 推測當事人的感受或情緒時所使用的語法。
  - 名詞 + にしたら
  - 親にしたら, 国立のほうか学費が安くていいです。
    - こくりつ, がくひ

5. 〜にすれば
  - 就...來想的話。同〜にしたら
  - 名詞 + にすれば
  - 社長にすれば, 社員の給料は安いほうがいいです。
    - きゅうりょう

6. 〜からして
  - (從某種觀點或立場)～來看的話，根據...。表示判斷的依據
  - 名詞 + からして
  - 今までの成績からして, 合格は不可能でしょう。
    - せいせき, ごうかく, ふかのう

7. 〜からすると 
  - (以某種觀點或立場)～來判斷的話
  - 名詞 + からすると
  - 教師の立場からすると, 学生は皆素直でいい子ばかりです。
    - きょうし, たちば, すなお

8. 〜にしては
  - 「以...來說」, 前文敘述的事實狀態/行為, 與預想的結果不同
  - 名詞 + にしては
  - 美和さんは日本人にしては, ずいぶん開放的な人ですね。
    - かいほうてき

### 行動基準的相關句型

1. 〜に沿って
  - 「遵照～」,「依循著～」表示依照著某種決定, 想法, 方針而做出行動的語法
  - 名詞 + にそって
  - 説明書に沿って, この棚を組み立てました。
    - たな, くみたて

2. 〜に基づいて
  - 以...為基礎, 以...為依據
  - 名詞 + にもとづいて
  - 現地の調査結果に基づいて, 論文を書きました。
    - げんち, ろんぶん

3. 〜をもとに(して)
  - 以...為素材做..., 通常後文接, 作る(つくる), 書く(かく), できる
  - 名詞 + をもとに(して)
  - 新聞に書いてある記事をもとに, ドラマの脚本を書いた。
    - きじ, きゃくほん

4. 〜のもとで
  - 在...之下, 通常用來表示在某種保護或影響之下做什麼事情
  - 名詞 + のもとで
  - 姉は有名な先生のもとで, 研究を続けたいと言っています。
    - けんきゅう

### 場合, 場面, 空間範圍的相關句型

1. 〜にあたって
  - 「值~之際」、「在~的時候」。屬於比較正式的語法 通常都用於正式場合 例如致詞,發表感言,演講等
  - 名詞/動詞辭書形 + にあたって
  - 卒業にあたって、家を出て独立しようと思っています。
    - そつぎょう, どくりつ

2. ~において
  - 「值~(狀態,場合)」「在~(狀態、場合)」。也是屬於比較正式的語法.
  - 名詞 + において
  - 今の我が国において、最大の問題は何でしょうか。
    - さいだい

3. 〜に際して
  - 「當~的時候」, 「值~之際」。通常是用來表示非日常性, 特別的事，表示正好碰到某種機會之意。
  - 名詞/動詞辭書形 + にさいして
  - この本の出版に際して、編集者の方々のご協力をいただきました。
    - しゅっぱん, へんしゅうしゃ, かたがた, きょうりょく

4. 〜から〜にかけて
  - 「從~(地方)到~(地方)」, 用來表示起點與終點不明確的範圍內
  - 名詞 + から + 名詞 + にかけて
  - 名古屋から京都にかけて、雪が降るそうです。
  - ふる

### 時間點, 期間範圍的相關句型

1. 〜うちに
  - 在...的時候
  - 名詞の/動詞普通形/イ形容詞/ナ形容詞な + うちに
  - デパートが閉まらないうちに, 急いで買い物に行きましょう。
    - しまらない

2. 〜最中に/〜最中だ
  - 正在...的當中, 正當...的盛況時
  - 名詞の/動詞ている形 + さいちゅうに/さいちゅうだ
  - 今, 会議をしている最中です。

3. 〜から〜まで
  - 從...(時間點), 到...(時間點). 強調時間的開始點和結束點
  - 名詞 + から + 名詞 + まで
  - 授業は九時から十二時までです。
    - じゅぎょう

4. 〜から〜にかけて
  - 從...(時間)到...(時間)。用來表示起點與終點不明確的期間範圍。
  - 名詞 + から + 名詞 + にかけて
  - 深夜から朝にかけて, 雨が降り続いた。
    - しんや, ふり

5. 〜たびに
  - 每次..., 總是...; 通常用來生活中經常反覆出現的狀態或事件, 即每當某種時候都會做某種動作時使用
  - 名詞の/動詞辭書形 + たびに
  - 日本へ行くたびに, 大学時代の先生を訪ねます。
    - じだい, たずねます

6. 〜にわたって
  - 整個(期間)都...; 經過～(期間)
  - 名詞 + にわたって
  - 彼は一年にわたって, 映画を撮り続けました。

7. 〜え通じて
  - 在(整個)～期間; 一直...; 在某個期間之內持續做某個動作
  - 名詞 + をつうじて
  - 高雄は一年を通じて, ずっと暖かいです。
  - たかお

### 事件的相互或連鎖關係的相關句型

1. 〜ば〜ほど / 〜なら〜ほど
  - 越...則越...
  - イ形容詞-い + ければ + イ形容詞 + ほど
  - ナ形容詞なら + ナ形容詞な + ほど
  - 動詞假定形 + 動詞辭書形 + ほど
  - 練習すればするほど, 上手になります。

2. 〜につれて
  - 隨著...而... / 越...則越...
  - 「Aにつれて, B」 表示隨著A事件的發展, B的部分也有所改變
  - 名詞/動詞辭書形 + につれて
  - 社会の変化につれて, 生活のスタイルも変わってきました。
    - かわって

3. 〜とともに
  - 和...一起, 與...的同時, 用來表達共同參與某事, 或兩事件同時發展
  - 名詞/動詞辭書形 + とともに
  - 家族とともに, 田舎で生活したいです。
    - いなか

### 事件添加性的相關句型

1. ～上に
  - 而且~, 再加上~; 表示在現有的狀態之下, 再加上, 重疊另一種狀態
  - 動詞普通形/イ形容詞/ナ形容詞な/名詞である + うえに 
  - 新しい携帯は機能が多い上に軽いので, たいへん便利です。
    - きのう
  
2. 〜に加えて
  - 「再加上～」, 「而且～」, 表達在某件事物之上再增添另一事物
  - 名詞 + にくわえて
  - 安い給料に加えて物価も上がり, 生活は苦しくなるばかりです。
    - きゅうりょう, ぶっか, あがり

3. 〜ばかりでなく
  - 「不僅如此～更～」, 「不僅～而且～」, 先敘述程度較輕微的事情, 然後再強調除此之外還有程度更高的事情
  - 動詞普通形/イ形容詞/ナ形容詞な/名詞 + ばかりでなく
  - この本は説明が簡単なばかりでなく, 内容も分かりやすいです。
    - ないよう

4. 〜に限らず
  - 「不僅僅只是～」, 「不光是～」, 同:ばかりでなく, 但較文言的表現
  - 名詞 + にかぎらず
  - 料理教室は女性に限らず, 男性も参加することができます。
    - きょうしつ, じょせい, だんせい

###　因果關係的相關句型

1. 〜せいで
  - 「因為...」,「由於...」, 表達不好的事情, 或非自己期望的事情發生的原因或責任所在
  - 名詞の + せいで
  - 大雨のせいで, 試合が中止になった。

2. 〜おかげで
  - 「託你的福...」,「幸虧...」, 表達因為某種原因, 理由, 而帶來好的, 或是期待的結果
  - 動詞た形/イ形容詞/ナ形容詞な/名詞の + おかげで
  - コンビニがたくさんできたおかげで, 生活が便利になりました。

3. 〜によって
  - 「因為...」,「由於...」, 表達因為某種原因, 而帶來不好的, 或是非期望的結果
  - 名詞 + によって
  - 台風の影響によって, 野菜の値段が上がりました。
    - たいふう, えいきょう

4. 〜につき
  - 「由於...」,「因為...」, 用於書信表現或是正式場合
  - 名詞 + につき
  - 事故につき, 通行止めです。
    - つうこう, どめ
  
### 時間前後, 動作先後的相關句型

1. 〜うちに
  - 「趁著...的時候」
  - 動詞普通形/イ形容詞/ナ形容詞な/名詞の + うちに
  - 若いうちに, 遠い国へ行ってみたい。
    - くに

2. 〜てはじめて
  - 在...之後終於..., 在事情發展到某個階段後才有某個想法或動作
  - 動詞て形 + はじめて
  - 病気になってはじめて, 健康が何よりも大切だと分かった。
    - けんこう

3. ～て以来(ていらい)
  - 自...以來, 從...之後, 表示某件事情發生之後就不再出現變化
  - 動詞て形 + 以来
  - 結婚して以来, 一度も海外旅行をしていません。
    - かいがい, りょこう

4. 〜上で(うえで)
  - 在...之後, 表示做完前項的動作之後, 依據前項的結果, 採取下一個步驟
  - 名詞の/動詞た形 + 上で
  - この書類はご確認の上で, サインしてください。
    - しょるい, かくにん

5. 〜たとたん
  - 剛...就..., 表示在前項的動作做完之後, 馬上就發生下一個動作
  - 動詞た形 + とたん
  - 布団に入ったとたん, 明日テストがあることを思い出した。

6. 〜次第(しだい)
  - 一...就馬上..., 在某件事情完成之後, 就馬上進行下一個動作
  - 名詞/動詞ます形 + 次第
  - 卒業次第, すぐ仕事を探そうと思います。
    - そつぎょう

### 事件的目的, 必然性的相關句型

1. 〜ため(に)
  - 為了...所以...; 表「目的」
  - 名詞の/動詞辭書形 + ため(に)
  - 結婚のため, 貯金しています。
    - ちょきん

2. ～ため(に)
  - 因為...所以...; 表「因果」
  - 名詞の/ナ形容詞な/イ形容詞普通形/動詞普通形 + ため(に)
  - 病気のため, 会社を休みました。
    - びょうき

3. ～ため(に)
  - 為了...(的利益); 表「利益」; 通常接續在表示人物或事物的名詞之後, 表示對其有益的意思
  - 名詞の + ため(に)
  - 家族のために働いています。

4. 〜はず
  - 應該是...; 推測,判斷; 根據客觀的因素來推論, 判斷, 認為理當如此的表現形式
  - 否定: はずがない, 〜ないはずです; 不可能..., 不會...
  - 名詞の/ナ形容詞な/動詞普通形 + はず
  - 昨日連絡したので, 中山さんも知っているはずです。

5. 〜べき
  - 應該...; 推測, 判斷; 說話者主觀上, 道義上的推論, 判斷
  - 〜べきではない; 不該...
  - 動詞辭書形 + べき
  - 交通ルールを守るべきです。
    - こうつう

## 3. 授受表現的相關句型
 
### 東西, 物品的授受表現

#### 以授予者的立場來表現的句型

1. 〜あげる
  - 授予者__給__接受者...
  - 授予者は + 接受者に + 名詞を + あげる
  - 私はクラスメートに誕生日のプレゼントをあげました。

2. 〜くれる
  - 授予者__給__我(我們)...
  - 授予者が + 我(我們)に + 名詞を + くれる
  - 兄が私に小遣いをくれました。
    - こづかい 

#### 以接受者的立場來表現的句型

1. 〜は〜に〜をもらう / 〜は〜から〜をもらう 
  - 接受者從授予者處得到...
  - 接受者は + 授予者に + 名詞を + もらう 
  - 接受者は + 授予者から + 名詞を + もらう 
  - わたしは兄(に/から)小遣いをもらいました。

2. 〜は〜にもらう / 〜は〜からもらう 
  - ...是從授予者處得到的
  - 物品は + 授予者に + もらう 
  - 物品は + 授予者から + もらう
  - プレゼントは友達 (に/から)もらいました

### 動作、行為的授受表現

#### 以授予者的立場來表現的句型 

1. 〜てあげる 
  - 授予者為接受者做...動作
  - 授予者は + 接受者に+ (名詞を) + 動詞て形+ あげる 
  - わたしは林君にチョコレートを買ってあげます。 

2. 〜てくれる 
  - 授予者為我(或我們)做...動作
  - 授予者が + 我(或我們)に + (名詞を) + 動詞て形 + くれる 
  - 姉がわたしに絵本を読んでくれました。

#### 以接受者的立場來表現的句型: 

1. 〜は〜に〜てもらう / 〜は〜から〜てもら 
  - 接受者從授予者處得到...動作 
  - 接受者は + 授予者に/から+ (名詞を) + 動詞て形 + もらう
  - わたしは母(に/から)おいしい料理を作ってもらいました 

2. ～は〜に〜てもらう / 〜は〜から〜てもらう
  - ...是從授予者處做...動作得到的
  - 物品は + 授予者に / から + 動詞て形 + もらう
  - パソコンは父に / から修理してもらいました。 
    - しゅうり

### 動作、行為授受表現的尊敬、謙讓語法

| 語彙 | 尊敬語 | 謙讓語 ｜語意 |
| -------- | ------- | ------| ----|
| あげる | さしあげる | やる | 給~人 | 
| くれる | くださる | - | ~人給(我) | 
| もらう | - | いただく | 得到、接受 |

#### 以授予者的立場來表現的句型

1. 〜てさしあげる
  - 授予者為接受者做...動作
  - 授予者は + 接受者に+ (名詞を) + 動詞て形+ さしあげる
  - わたしは部長に荷物を持ってさしあげました。

2. 〜てくださる
  - 授予者為我(或我們)做...動作
  - 授予者が + 我(或我們)に + (名詞を) + 動詞て形 + くださる
  - 先輩がわたしに仕事を説明してくださいました。
 
#### 以接受者的立場來表現的句型

1. 〜ていただく
  - 接受者從授予者處得到...動作
  - 接受者は + 授予者に / から + (名詞を) + 動詞て形 + いただく
  - わたしは先生に/から日本語の読み方を教えていただきました。

2. 〜は〜に〜ていただく / 〜は〜から〜ていただく
  - ...是從授予者處做...動作得到的
  - 物品は + 授予者に / から + 動詞て形 + いただく
  - かばんは課長に / から買っていただきました。

## 4. 被動(受身), 使役, 使役被動的相關句型

### 動詞的被動句型

#### 動詞被動(受身)句型 I

- 被動(受身)句型 I, 是以有情物(人物、動物)為主語,除了幾個例外的語彙之外, 絕大部分的被動句都是說話者(A)要表達因動作者(B)的行為而感到受害或困惑的情緒
- 例外語彙有「ほめる」(稱讚)、「誘う(さそう)」(邀約)、「助ける(たすける)」(幫助)、 「頼む(たのむ)」(拜託)、「招待する(しょうたいする)」(招待)、「紹介する(しょうかい)」(介紹)、「案内する(あんないする)」(指引)

1. AはBに〜れる / 〜られる
  - A被B...
  - A(主語)は + B(動作者)に + 動詞ない形 + れる / られる(被動句)
  - わたしは友達に来られて, 勉強ができませんでした。
    - こられて
2. AはBに名詞を〜れる / 〜られる
  - 屬於A的...被B...; A遭受了B的行為或動作
  - A(主語)は + B(動作者)に + 名詞を+ 動詞ない形 + れる/られる(被動句)
  - わたしは友達に京都のお寺を案内されました。
    - おてら

#### 動詞被動(受身)句型 II

- 被動(受身)句型 II ,則是以人的動作、行為的內容(物) 為主語的語法表現。

1. Aは〜れる / 〜られる
  - A是被...
  - A(事物)は + 動詞ない形 + れる / られる (被動句)
  - 日本では、入学式は每年の4月に行われます。
    - にゅうがくしき, まいとし, おこなわれ

### 動詞的使役句型

- 使役句型,顧名思義即是「使人勞役」的語法。
- 句中必然有使役者(A)及動作者(B)的存在
- 通常使役者(A)的年齡、階級等,高於動作 (B) 
- 動詞本身含有使役意義的語彙(例:泣かす(使~哭泣)、起こす(使~立起、喚醒)

1. AはBを〜せる / 〜させる
  - A使B...; A讓B...; A叫B...
  - 使役者(A)は + 動作者(B)を + 自動詞ない形 + せる/させる
  - 先生は学生を立たせました。(立つ, たつ)
  
2. AはBに名詞を〜せる / 〜させる
  - A使B...; A讓B...; A叫B...
  - 使役者(A)は + 動作者(B)に + 名詞を+ 他動詞ない形 + せる/させる
  - 社長はわたしにレポートを書かせました。(書く)

### 動詞的使役被動句型

- 使役被動句型是以「被指使者」的立場所看待的使役動作
- 「某人被指使去做某事」
- 使役被動句型是將使役句型改為被動

1. AはBに~ (さ)せられる
  - A被B要求做...
  - A(被使役者)は + B(使役者)に + 名詞を + 動詞ない形 + (さ)せられる
  - 父は医者にお酒をやめさせられました。(やめる)
  - わたしはセールスマンに本を買わせられました。
    - 買わされました
  - 弟は母に勉強をさせられました。(する)

## 5. 假設、假設逆態的相關句型

### 之前

- 初級日語中, 以「AとB」、「AたらB」、「AばB」、「AならB」來表示假設(條件)。
  - お金があれば、ぜひ日本へ行ってみたいです。
  - 京都へ旅行に行くなら、 九月が一番いいと思います。
- 假設(條件)的逆態句則以「〜ても」、「〜のに」表示・
  - お金があっても、旅行に行きません。
  - お金がないのに、たくさん買い物しました。

### 中級

1. ～とすると
  - 如果...; 假如...的話。通常以「Aとすると、B的」句型出現
  - 句首經常會伴隨強調語彙的「もし」出現, 表達假如A的敘述是事實的話,則B的現象、狀態、結果等會同時出現。
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞だ / 名詞だ + とすると
  - 日本語ができないとすると、日系企業で働くのは辛いでしょう。
    - にっけい, きぎょう, つらい

2. 〜としたら
  - 如果...; 假如...的話
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞だ / 名詞だ + としたら
  - 生まれ変わるとしたら、人間以外の何になりたいですか。
    - うまれ, にんげん

3. 〜とすれば
  - 如果...; 假如...的話
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞だ / 名詞だ + とすれば
  - 不況が続くとすれば、生活はますます苦しくなるでしょう。
    - ふきょう, せいかつ

4. 〜としても
  - 即使...也...; 條件(假設)句型的逆態語法
  - 以過去式「〜たとしても、B」: 表示即使是A事態成立的狀態下B也...
  - 以現在式「〜としても、B」: 表示A和B成立時間點相同,是將來要發生的動作狀能
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞(だ) / 名詞(だ) + としても 
  - これからタクシーに乗ったとしても、間に合いそうもない。
    - まにあい

5. 〜にしても
  - 即使是...的話也...; 條件(假設)句型的逆態語法
  - 「Aにしても、B」, 表達即使A是事實也可能有B的狀況出現
  - 動詞普通形 / イ形容詞普通形 / 名詞である / ナ形容詞である + にしても
  - お金がないにしても、人のお金を盗むのはいけないことです。
    - ぬすむ

## 6. 逆接的相關句型

### 之前

- 以「〜が」、「〜けれども」、「〜ても」; 來表示前、後文語意或語氣相違逆的逆接語法
- 雖然...但是...
- 日本料理はおいしいですが、高いです。
- 時間があっても、お金がなければ、旅行に行くことができません。

### 中級

1. 〜のに
  - 明明...卻...; 
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞な / 名詞な + のに
  - 雨が降っているのに、傘も差さないで出かけました。
    - ささない
  
2. 〜ながら/〜ながらも 
  - 雖然...但是...
  - 表達雖然A的敘述是事實但是卻是B的狀態
  - 動詞ます形 / イ形容詞 / ナ形容詞であり / 名詞であり + ながら/ ながらも
  - 彼にお礼を言おうと思いながら、とうとう言うのを忘れてしまいました。
    - おれい

3. 〜といっても
  - 雖然...但是卻...
  - 表達雖然有A的事實, 但是程度卻不如想像中來得高
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞普通形 / 名詞普通形 + といっても
  - 日本語ができるといっても、簡単な会話だけで、まだまだ勉強しなければなりません。

4. 〜くせに
  - 雖然...卻...
  - 說話者認為A是不好的、或是不對的,因此以責備、不滿的情緒表達心中的想法
  - 動詞普通形 / イ形容詞普通形 / ナ形容詞な / 名詞の + くせに
  - 知っているくせに、黙っているなんて、ひどい人だ。
