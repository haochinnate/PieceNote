- CLI Agentic Coding Assistant
  - Gather context
  - Formulate a plan
  - Take an action


## claude.md

- 跨工作階段記憶 (Memory across sessions) (claude.md):
  - 在 Claude.md 檔案 (files) 中，你可以定義你的風格指南 (style guidelines) 和常用指令 (common commands)。
  - 你將會了解更多關於 Claude.md 檔案的不同記憶位置 (memory locations)。
  - 啟動時，Claude.md 檔案會被 自動 (automatically) 載入到 Claude Code 的上下文 (context) 中。

- 對話紀錄 (Conversation history):
  - 對話紀錄 (Conversation history) 會自動儲存 (stored) 在你的本機 (locally) 電腦上。
  - 你可以選擇清除 (clear) 目前工作階段 (current session) 的對話。
  - 你可以選擇在未來的工作階段 (future session) 恢復 (resume) 對話。
  - 過去的對話 (Past conversations) 不會自動包含在上下文 (context) 中。你需要手動 (manually) 要求 Claude Code 繼續先前的對話 (previous conversation)。

```shell
claude
# 輸入後, 會詢問是否相信此資料夾




```
